# Weather App 

## התקנה והרצה

### 1. שכפול הפרויקט
```bash
git clone https://github.com/MiriRadunsky/weather-app.git
cd weather-app
```

### 2. התקנת תלויות
```bash
npm install
```

### 3. הגדרת משתני סביבה
שנו את שם הקובץ מ-`.env.example` ל-`.env`:

```bash
mv .env.example .env
```

### 4. הרצת הפרויקט
```bash
npm run dev
```
